前言

  • 想做ppt，但几次ppt演示时出现兼容问题，于是想尝试格式稳定的beamer 做ppt 。
  • 本机空间不足，apt-get install 命令无法指定安装路径，于是挂载了一个硬盘，手动下载后安装，空间充足可sudo apt install texlive-full texstudio 一行命令解决。
